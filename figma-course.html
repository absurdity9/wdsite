Web Design using Figma Course

I taught a course part time at Middlesex University (London) and I plan to share it here so that non Designers can learn to use Figma to make basic websites and apps.

Lesson 1 slides

This lesson introduces the structure of the course, the final project, concepts for thinking about web design, tooling, Figma, typography styles, how to use the Text and Frame tools. View the slides.

Lesson 2 slides

This lesson covers font pairing, checking layers and naming, looking for inspiration, user journey mapping, coming up with a color palette, wireframes and the module structure of websites and working with images in Figma. View the slides.

Lesson 3 slides

This lesson covers creating a moodboard for inspiration, using autolayout and the importance of spacing in design. View the slides.

Lesson 4 slides

This lesson covers layouts on websites, basic prototyping, animations in Figma and competitve analysis. View the slides.

Lesson 5 slides

This lesson covers how to make use of trends in visual design, working with vectors in Figma, using GIFS and user testing. View the slides.
